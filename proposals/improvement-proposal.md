作成日: 2026-03-13
ステータス: **クライアント確認待ち**

## エグゼクティブサマリー

menskajiyaru.com は「男性の体験談・副業・AGA治療」等を発信するWordPressメディアです。
SEOツール・Google Analytics・Yoast SEO等の基盤は整っている一方で、**法的リスク・パフォーマンス・SNS集客・CV導線**に複数の重大な課題が見つかりました。

優先度の高い順に対応することで、検索流入の維持・増加と収益化の強化が期待できます。

## P0：即時対応（法的リスク・重大欠落）

### 🔴 P0-1: プライバシーポリシーページの作成・フッターリンク設置

**問題**: サイト全体にプライバシーポリシーが存在しない。Google Analytics・アフィリエイト利用サイトには**法律上（個人情報保護法・景品表示法）必須**。ASPの審査でも要求される。

**対応内容**:
1. WordPressで「プライバシーポリシー」固定ページを作成
2. Google Analytics・Cookieの利用について記載
3. アフィリエイト広告である旨の開示文を記載
4. フッターにリンクを設置

**担当**: PM確認 → フロントエンドエンジニア実装

---

### 🔴 P0-2: XウィジェットへのSNSアカウントURL設定

**問題**: サイドバーに「x(twitter)」というタイトルのウィジェットが表示されているが、URLが設定されておらず空のウィジェットになっている。

**対応内容**:
- WordPress管理画面 → 外観 → ウィジェット → 「x(twitter)」ウィジェットにXアカウントURL（https://x.com/アカウント名）を入力

**担当**: クライアント（Xアカウント名の確認が必要）

## P1：高優先度（SEO・UX・収益に直結）

### 🟠 P1-1: ホームページ meta description の改善

**問題**: 現在の meta description が17文字と極端に短く、Googleに書き換えられている可能性が高い。CTRの損失。

**現状**: `男の家事育児メディア` (17文字)
**改善案**: `40代男性の副業・AGA治療・育児・料理の体験談を発信するメディア。経営者やる夫が実際に試した情報をリアルに届けます。` (60文字程度)

**対応**: Yoast SEO → 検索の見え方 → サイトタイトル/メタ説明を更新

---

### 🟠 P1-2: URL構造の改善（新規記事から適用）

**問題**: 現在のURL構造が `/2026/03/11/日本語スラッグ/`（日付+日本語）。SEO上、英数字スラッグが推奨。既存記事の変更は301リダイレクト対応が必要なため**新規記事から**適用。

**推奨構造**: `/fukugyo-hajimenkata/`（英数字のみ、カテゴリなし）

**対応**:
1. WordPress → 設定 → パーマリンクを「投稿名」に変更
2. 既存記事は変更せず、新規記事から英数字スラッグを設定
3. 既存記事のスラッグ変更は段階的に（301リダイレクト必須）

---

### 🟠 P1-3: 記事内 SNS シェアボタンの設置

**問題**: 体験談・AGA記事などSNS拡散に適したコンテンツがあるが、記事内にシェアボタンが一切ない。

**対応**:
- プラグイン導入: `AddToAny Share Buttons`（無料・軽量）
- 表示ボタン: X・Facebook・LINE・はてなブックマーク
- 配置: 記事タイトル直下 + 記事末尾

---

### 🟠 P1-4: 目次（TOC）の設置

**問題**: 長文記事（2000字以上）に目次がない。ユーザーが読みたい箇所に移動できず、直帰率が上がりやすい。

**対応**:
- プラグイン導入: `Easy Table of Contents`
- H2・H3を自動検出して記事冒頭に目次を表示

---

### 🟠 P1-5: フッターの整備

**問題**: フッターが完全に空。プライバシーポリシー・お問い合わせ・プロフィールへのリンクがない。

**対応**:
- プライバシーポリシー・サイトについて・お問い合わせのリンクをフッターメニューに追加
- WordPress → 外観 → メニュー → フッターメニュー作成

---

### 🟠 P1-6: og:description の修正（「Read More →」混入問題）

**問題**: SNSシェア時のOGP説明文に「Read More →」が混入している。

**対応**: 各記事のYoast SEO設定 → 「ソーシャルの説明」フィールドに明示的な説明文を入力。または記事抜粋フィールドを適切に設定。

---

### 🟠 P1-7: twitter:site / twitter:creator メタタグ設定

**問題**: XのCard表示時にアカウントが紐づかないため、フォロワー獲得機会を失っている。

**対応**: Yoast SEO → ソーシャル → Twitter/X → サイトのTwitterユーザー名に `@アカウント名` を入力

---

### 🟠 P1-8: ページキャッシュプラグインの導入

**問題**: DOMContentLoaded 2,498ms・Load 2,850ms と遅い。ページキャッシュがない可能性が高い。

**対応**:
- `WP Super Cache`（無料・軽量）または `W3 Total Cache` を導入
- Autoptimize と組み合わせてCSS/JS遅延読み込みを有効化

---

## P2：中優先度（品質・信頼性の向上）

### 🟡 P2-1: 著者プロフィールボックスの設置

**問題**: 記事末尾に著者情報がなく、E-E-A-T（専門性・信頼性）が弱い。

**対応**:
- WordPress → ユーザー → プロフィール にやる夫の経歴・経験を記入
- テーマまたは `Simple Author Box` プラグインで記事末尾に表示

---

### 🟡 P2-2: お問い合わせフォームの設置

**問題**: 企業からの案件依頼・読者からの問い合わせ手段がない。サイト成長時の機会損失。

**対応**:
- `Contact Form 7` プラグインで問い合わせページを作成
- フッター・ヘッダーメニューにリンクを追加

---

### 🟡 P2-3: 画像の最適化（WebP・lazy loading・alt属性）

**問題**:
- 全画像がWebP形式ではない（推定）
- lazy loading が0枚（ページ全体の画像を一括読み込み）
- alt属性なし画像が4枚

**対応**:
- 新規アップロード画像は WebP 形式に変換
- `EWWW Image Optimizer` プラグインで既存画像を一括変換・圧縮
- WordPress設定で lazy loading を有効化（WP 5.5以降はデフォルト有効のはずだが要確認）
- alt属性なし画像に適切なテキストを追加

---

### 🟡 P2-4: 関連記事ウィジェットの設置

**問題**: 記事末尾・サイドバーに関連記事がなく、回遊率が低い。

**対応**:
- `Related Posts by Jetpack` または テーマ設定で記事末尾に関連記事を表示（3〜5記事）

---

### 🟡 P2-5: アフィリエイトリンクの整備・増加

**問題**: 副業・転職・AGA等の収益化しやすいテーマを扱っているが、アフィリエイトリンクがDMMクリニックの1本のみ。

**対応**:
- 副業記事: クラウドワークス、ランサーズ、ストアカ のアフィリエイトリンク設置
- 転職記事: リクルートエージェント、doda 等
- AGA記事: 既存DMMクリニックに加え、AGAスキンクリニック等を追加

---

### 🟡 P2-6: カテゴリの統廃合

**問題**: 11カテゴリが乱立し、「政治経済」「ガジェット」がサイトコンセプト（男性の体験談メディア）と乖離。

**対応**:
- コアカテゴリ（副業・転職・育児・料理・AGA・体験談）に絞る
- 「政治経済」「ガジェット」は非表示または削除検討

---

### 🟡 P2-7: セキュリティヘッダーの設定

**対応**: `.htaccess` に以下を追加:
```
Header set X-Content-Type-Options "nosniff"
Header set X-Frame-Options "SAMEORIGIN"
Header set Referrer-Policy "strict-origin-when-cross-origin"
```

---

### 🟡 P2-8: バックアップ設定

**対応**: `UpdraftPlus` プラグインで週次自動バックアップ（保存先: Google Drive）

## P3：低優先度（長期施策）

### 🟢 P3-1: LINE公式アカウント または メールマガジンの導入

読者リスト獲得のため、LINE公式アカウントまたはメルマガ登録フォームを設置。

### 🟢 P3-2: 記事末尾「XでフォローCTA」の追加

各記事末尾に「XでやR夫をフォローする」テキストリンクを追加してSNSフォロワーを増やす。

### 🟢 P3-3: はてなブックマークボタンの設置

日本語ブログ系コンテンツははてブからの流入が見込める。P1-3のシェアボタンに含めて設置。

### 🟢 P3-4: Cloudflare の導入

CDN・DDoS対策・パフォーマンス改善を一元化。無料プランでも効果大。

### 🟢 P3-5: Google Search Console での sitemap 登録確認

Yoast SEOの sitemap_index.xml が GSC に登録されているか確認・未登録なら登録。

## 転職ガイドページ群 追加提案

> 対象: `menskajiyaru.com/年代・家族構成別転職市場ガイド/` およびサブページ全21ページ
> 位置づけ: 転職アフィリエイトサイト化予定の固定ページ群

### 現状評価
コンテンツ品質・デザインは**高水準**。年代×家族構成の細分化は差別化戦略として有効。
しかし**アフィリエイトリンクが全ページで0件**のため収益がゼロの状態。

### 🔴 TG-P0（即時）: meta description の手動設定

**問題**: 全21ページで meta description がナビメニューのテキストになっている（自動生成バグ）。
Yoast SEO → 各固定ページ → 「検索の見え方」→ メタ説明文に手動入力する。

**各ページの説明文方針（120文字以内）**:
- ハブ: `20代〜60代・独身/既婚/子どもあり別に転職戦略を徹底解説。首都圏・関西・地方都市の最新データで「どこで・どう働くか」を設計します。`
- サブページ例（40代子あり）: `40代・子どもあり世帯の転職。高校受験への影響・転居タイミング・移住支援金・年収設計を都市別データで解説。`

### 🔴 TG-P1（高優先）: アフィリエイトリンクの設置

**問題**: 転職アフィリエイトサイトなのに転職エージェントへのリンクが1件もない。

**設置箇所（各サブページ共通）**:
1. **記事冒頭**（ファーストビュー直下）: メインCTAボックス1個
2. **記事中盤**（3〜4セクション目付近）: インライン推奨リンク
3. **まとめセクション直後**: メインCTAボックス + サブCTA

**推奨アフィリエイト（年代別）**:

| ターゲット | 推奨サービス | 理由 |
|-----------|-------------|------|
| 20〜30代 | リクルートエージェント / doda | 求人数最大・成功率高 |
| 40代 | ビズリーチ / JACリクルートメント | ハイクラス・管理職向け |
| 50〜60代 | マイナビシニア / シニアジョブ | シニア専門 |
| 全年代 | doda / マイナビ転職 | 幅広いカバレッジ |

**CTAボックス実装例**:
```html
<div class="affiliate-cta">
  <p class="cta-label">👔 この記事を読んだあとの次のステップ</p>
  <h3>まずは無料で転職相談してみる</h3>
  <a href="[アフィリエイトURL]" class="cta-btn" target="_blank" rel="noopener">
    リクルートエージェントに無料登録する →
  </a>
  <p class="cta-note">※ 登録・相談完全無料。転職しなくてもOK</p>
</div>
```

### 🟠 TG-P1: ハブページのタイトルタグ修正

**問題**: `年代・家族構成別転職市場ガイド - メンズ家事やるメンズ家事やる`（「メンズ家事やる」が2回）

**修正後**: `年代・家族構成別 転職市場ガイド 2026年版 | メンズ家事やる`

### 🟠 TG-P1: H2構造の見直し

**問題**: 各サブページのH2が「目次」「まとめ」「出典一覧」のみ。本文コンテンツはH3で表現されているためSEO評価が弱い。

**対応**: 現在のH3（「転職を検討するきっかけ」「受験と転居」等）をH2に昇格し、論理的見出し構造に統一。

### 🟡 TG-P2: URLスラッグの英数字化（301リダイレクト必須）

| 現在（日本語） | 変更後（英数字） |
|--------------|----------------|
| `/年代・家族構成別転職市場ガイド/` | `/tenshoku-guide/` |
| `/40代-x-子どもあり-x-転職ガイド/` | `/tenshoku-40s-with-kids/` |
| `/30代-x-子どもあり-x-転職ガイド/` | `/tenshoku-30s-with-kids/` |
| `/30代-x-夫婦-子なし-x-転職ガイド/` | `/tenshoku-30s-couple/` |

### 🟡 TG-P2: 転職エージェント比較表の追加

各年代ページに「このターゲットに合う転職サービス比較」テーブルを追加。
比較軸: 求人数・対応年代・得意分野・登録時間・コンサルタントの質・報酬単価

### 🟢 TG-P3: og:type の修正

固定ページの `og:type` を `article` → `WebPage` に変更（Yoast SEO設定）。
