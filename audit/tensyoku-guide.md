転職アフィリエイトサイトとして設計されているにも関わらず、**全ページにわたってアフィリエイトリンクが1件も設置されていない**。

- 外部リンク先: wordpress.org / yoa.st（Yoast） / search.google.com（GSC） / developers.facebook.com / developers.google.com のみ
- 転職エージェント・転職サービスへのリンク: **ゼロ**

### 機会損失
以下のような高単価アフィリエイト（1件2,000〜10,000円）を設置できるタイミングが随所にある:

| 地方移住×転職 | 高専ネットワーク就職・地方特化型転職サービス |

## 5. meta description 自動生成バグ

### 問題の根本原因
Yoast SEOの抜粋生成ロジックが、**固定ページのカスタムHTML（ナビゲーションメニュー）のテキストをページ本文の先頭として誤認識**している。

- Yoastの自動抜粋: ページ内HTMLの冒頭テキストノードを取得
- 固定ページのカスタムHTML内ナビ（「転職市場ガイド 20代 30代...」）が先頭に来る
- ナビのテキストがそのまま meta description / og:description に設定される

### 影響
- Google検索結果でのスニペットが崩れる
- SNSシェア時のカード説明文が意味不明になる
- 全22ページ共通の問題

### 修正方法
各固定ページのYoast SEO設定 → 「検索の見え方」タブ → メタ説明文欄に**手動で説明文を入力**（優先適用される）

## 6. URL構造の問題

### 現状
| ページ | URLスラッグ |
|--------|------------|
| ハブ | `/年代・家族構成別転職市場ガイド/` |
| 40代子あり | `/40代-x-子どもあり-x-転職ガイド/` |
| 全サブページ | 日本語混在 |

### 問題
- 日本語URLはエンコード時に長くなり、被リンク・共有時に崩れやすい
- SEO的には英数字スラッグが推奨
- 現状は `%e5%b9%b4%e4%bb%a3...` と表示され見栄えが悪い

### 推奨スラッグ例
| 現在 | 推奨 |
|------|------|
| `/年代・家族構成別転職市場ガイド/` | `/tenshoku-guide/` |
| `/40代-x-子どもあり-x-転職ガイド/` | `/tenshoku-40s-with-kids/` |
| `/30代-x-夫婦-子なし-x-転職ガイド/` | `/tenshoku-30s-couple/` |

※ 変更時は301リダイレクト必須

## 8. 改善施策（優先度別）

### P0（即時）
- **全ページの meta description を手動入力**: Yoast SEO → 各固定ページの説明文欄に入力（ナビテキスト混入バグ対応）

### P1（高優先）
- **アフィリエイトリンクの設置**: 各ページ末尾の「まとめ」直後に転職エージェントCTAボックス追加
  - 推奨: リクルートエージェント（成果報酬型・高単価）
  - 推奨: doda（ターゲット層に合致）
  - 推奨: ビズリーチ（40代以上・ハイクラス向け）
- **H2→H3の構造見直し**: 現在H3のコンテンツセクションをH2に昇格（各ページ本文のSEO評価向上）
- **ハブページのTitleタグ修正**: 「メンズ家事やる」の重複を除去

### P2（中優先）
- **スラッグの英数字化**: 301リダイレクトと合わせて段階的に変更
- **og:type修正**: 固定ページは `WebPage` に変更
- **CTAボタンの統一デザイン**: 全ページ共通のアフィリエイトCTAコンポーネント作成
- **著者プロフィールの追加**: やる夫（経営者）の転職経験・実績を記述

### P3（低優先）
- **転職エージェント比較表の追加**: 各年代ページに「このターゲットに合う転職サービス比較」セクション追加
- **内部リンクの強化**: 特集コンテンツ ↔ 年代ページ間のリンク増加
- **パンくずリストの実装**: ハブ → サブページ の階層表示
